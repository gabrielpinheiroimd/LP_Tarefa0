# LP_Tarefa0
Repositório da tarefa 0 de Lp 1


## Arquivos

* `conta_string.c` – Lê uma string e conta quantos caracteres ela possui.
* `fatorial.c` – Calcula o fatorial de um número inteiro.
* `num_primo.c` – Verifica se um número inteiro é primo.
* `temperatura.c` – Converte temperatura entre Celsius e Fahrenheit.
* `soma_array.c` – Lê 5 números inteiros, armazena em um array e calcula a soma.
* `print_1aN.c` – Lê um número inteiro positivo N e imprime os números de 1 até N.

## Compilação

Para compilar os programas, utilize o compilador GCC no terminal.

Pode ser necessário acessar a pasta correta caso use o codespace do GitHub, para isso use o comando

```bash
cd LP1_Tarefa0 

```

```bash
gcc conta_string.c -o conta_string
gcc fatorial.c -o fatorial
gcc num_primo.c -o num_primo
gcc temperatura.c -o temperatura
gcc soma_array.c -o soma_array
gcc print_1aN.c -o print_1aN
```

## Execução

Após compilar, execute o programa desejado:

```bash
./conta_string
./fatorial
./num_primo
./temperatura
./soma_array
./print_1aN
```

Cada arquivo `.c` é independente e pode ser compilado e executado separadamente.
